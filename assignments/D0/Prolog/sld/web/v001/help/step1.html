Schritt 1 - Programm eingeben

Um SLD-Bäume zu berechnen müssen Sie zuerst ein Prolog-Programm eingeben. Dazu klicken Sie rechts oben (oder hier) auf den Button "bearbeiten".

bearbeiten

Es öffnet sich ein Fenster indem Sie ein Prolog Programm eintippen können. Durch Klicken auf den Button "übernehmen", wird das eingegebene Programm an den Server gesendet und dort geprüft. Falls das Programm Syntaktisch korrekt ist, wird das Programm-Listing auf der rechten Seite aktualisiert. Tritt ein Fehler auf, so wird rechts "fail." angezeigt. In diesem Fall sollten Sie Ihr Programm auf eine fehlerhafte Syntax prüfen.

Wurde Ihr eingegebenes Programm korrekt übernommen, können Sie das Eingabefnster mit Klick auf schließen.
