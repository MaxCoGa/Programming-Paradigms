Question 1. Solution orientée-objet (Java) [3 points]

 

Créer les classes nécessaires pour résoudre le problème de transport.


Votre programme doit être une application appelée findOptimalTransport prenant en argument le fichier de données d’entrée. Notez que le but ici est de concevoir une solution qui tire profit du paradigme étudié; il n'est pas nécessaire de traiter les cas d'exception (par exemple, vous pouvez assumer que le fichier sera toujours correctement formatté).


En plus du code source, vous devez aussi remettre un diagramme de classe UML montrant toutes les classes, leur attributs, méthodes et associations. Vous ne pouvez pas utiliser de méthodes statiques (à l’exception du main).
